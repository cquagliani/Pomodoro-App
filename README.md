# Pomodoro-App

This is a native iOS app developed in SwiftUI, that helps users be more productive by using the Pomodoro technique. With custom appearance settings, the minimal interface and live activities reduce distractions while keeping the user informed of their session progress.

## Features

- Customizable timer lengths
- Appearance settings (color mode, focus session emoji, break session emoji, etc.)
- Live activity on the lock screen to track session progress at a glance
- Dynamic island to track session progress at a glance
- Push notifications after each round ends.

## Frameworks
- SwiftUI
- UIKit
- ActivityKit
- WidgetKit
- UserNotifications
  
## Installation

- Clone the repository: git clone https://github.com/cquagliani/Pomodoro-App.git
- Navigate to the project directory.
- Open the project in Xcode.
- Build and run the application in the simulator or on your device:
  - Select your target device from the dropdown in Xcode.
  - Press `Cmd + R` to build and run the application.
